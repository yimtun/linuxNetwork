删除 5 6

使用编号删除

iptables -D INPUT 5 -t filter

使用语句删除

iptables -D INPUT   -t filter -s 192.168.1.6 -j DROP
iptables -D INPUT   -t filter -s 192.168.1.5 -j DROP
